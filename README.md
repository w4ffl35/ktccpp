# Keyword Transposition Cipher C++

This is a basic keyword transposition cipher written in C++

## Prerequisites

- CMake 3.22 or higher
- C++23 compiler

## Build

```bash
bash build.sh
```

## Run

```bash
./build/ktccpp
```

You be asked for a secret key, then a message, and then you will presented with the option to encrypt or decrypt.
